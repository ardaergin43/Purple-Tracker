# PurpleTracker

PurpleFocus is a Python-based program designed specifically for users with purple color blindness. It is **not intended for gaming or gaining advantages** over other users. The application detects purple tones on your screen and automatically moves the mouse cursor over them.

## üñ±Ô∏è How It Works
1. Run `main.py` with Python installed on your system.
2. Press the **C key** on your keyboard to activate the program.
3. When active, the mouse cursor will automatically move to and stay over any purple color detected on your screen, with zero delay.
4. If disabled, the mouse will ignore purple colors entirely.
